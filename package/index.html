Process History

A free and portable process database
New 64bit version in available. Featuring: Process Command lines, easier navigation and jump to most recent processes, filtering on executable.

Download for Windows 7 and above, Apr 2018
64 bit - The 64 bit logger will log both 32 bit & 64 bit processes and will only run on 64bit Windows.
32 bit- The 32 bit logger will log only 32 bit processes running on 32 & 64 bit windows and is no longer updated.
